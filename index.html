03 / Bootcamp · Django

Alke Wallet — Django

Billetera digital full stack con backend en Django

Evolución del proyecto Alke Wallet hacia una aplicación web full stack construida con Django. Implementa autenticación de usuarios, modelos relacionales (OneToOne, ForeignKey, ManyToMany), operaciones CRUD completas, consultas ORM avanzadas y SQL raw, además de un dashboard con reportes financieros.

  • Autenticación completa con LoginRequiredMixin y tokens CSRF
  • Modelos relacionales: Cliente, Cuenta, Transacción y Beneficiario
  • Consultas ORM avanzadas: filter, exclude, annotate, aggregate
  • Dashboard con reportes, saldos y resumen de transacciones
  • Configuración dual: SQLite (desarrollo) y PostgreSQL (producción)
